# logsディレクトリ

このディレクトリにはアプリケーションログが出力されます。

## 注意事項

1. このディレクトリには書き込み権限が必要です
2. デプロイ時に必ずこのディレクトリを作成してください
3. 本番環境では定期的なログローテーションを推奨します

## ログファイル

- `app_YYYYMMDD.log` - アプリケーションログ

## 権限設定（Linux/Unixの場合）

```bash
chmod 755 logs/
```

## 権限設定（Windowsの場合）

IISのアプリケーションプール（通常はIIS_IUSRS）に書き込み権限を付与してください。
